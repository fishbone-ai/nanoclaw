# M1 — onboarding opener

Copy the text inside the fence. Send as the first message to each new participant via WhatsApp. Bot does **not** send this — the operator does, directly.

```
היי! אני פיט 💪 המאמן האישי שלכם לאימונים ותזונה.

אני יכול לעזור לכם:
1. לעקוב אחרי מה שאתם אוכלים 🍽️
2. לענות על כל שאלה באימונים 🙋
3. לבנות תוכנית אם אין לכם כזו 🏋️
4. לשמוע, לתמוך ולהיות איתכם בתהליך 🤝

אתם שולטים בחוויה, רק תגידו לי איך אתם מעדיפים לעבוד ואני אתאים.

אני עדיין לומד ומשתפר, אז אם משהו לא עובד, אל תתביישו תגידו לי 🙏

בואו נתחיל! איך קוראים לכם?
```
