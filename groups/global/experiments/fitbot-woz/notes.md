# Running notes

Operator log: observations, tone calibrations per user, things that worked, things that didn't. Update anytime something notable happens. This is the qualitative analysis material for the end-of-week review.

## Format

```
YYYY-MM-DD [user-name | global] — observation
```

## Prompts to answer at end of week

- Where did the bot's Hebrew feel off? Which specific phrases?
- Which participants disengaged and why (as far as you can tell)?
- Which interactions felt "too much bot" vs. "actually like a friend"?
- What did users ask for that the prompt didn't handle well?
- Meal logging accuracy on Israeli foods (shakshuka, pita, etc.) — where did it fail?
- Any tone-escape requests? What did users ask for instead of the default?
- Which prompt variant did you pick (Hebrew / English) and why?
