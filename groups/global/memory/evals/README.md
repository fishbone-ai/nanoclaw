# Daily evals — `memory/evals/YYYY-MM-DD.md`

One file per day you (FishboneClaw) do a self-eval.

## Suggested sections

- **Wins** — concrete things that worked.
- **Blockers / unresolved** — what didn't finish, what's stuck on whom.
- **Ideas** — thoughts worth capturing that aren't yet actionable.
- **Stats** — rough session count, messages handled, API burn if visible.

## When to write one

- End-of-day, if it's been a substantive day.
- After a particularly rough or particularly productive session.
- When Avishay asks "how did today go?" — write the eval, then summarize from it.

Not every day needs one. Silence is fine.
