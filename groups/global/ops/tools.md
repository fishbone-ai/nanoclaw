# Tools

What we're currently using and why.

## Active

| Tool | Purpose | Notes |
|------|---------|-------|
| GitHub | Code and ops repo hosting | fishbone org |
| Claude Code | AI-assisted operations, writing, research | Primary way we work with this repo |
| VS Code | Editor when needed | Mostly use Claude Code directly |
| `strategy/assumptions/cli.py` | CLI for reading/updating assumptions spreadsheet | Wraps Google Sheets via `gws` |
| `gws` (Google Workspace CLI) | Read/write Google Sheets, Drive | `npm install -g @googleworkspace/cli` |

## Evaluating

- CRM (TBD) -- need something lightweight for prospect tracking
- Email tooling -- for outreach at slightly higher volume

## Principles

- Start with the simplest tool that works
- Don't adopt tooling until the pain of not having it is obvious
- Prefer tools that play well with Claude Code and plain text workflows
